## Monorepo Template with NextJs and NestJS


#### How clone project

```sh
git clone https://github.com/teigorme/monorepo-template.git
```


#### How install lib of project

##### Install lib for web
```sh
cd src/apps/web && yarn 
```

##### Install lib for api
```sh
cd src/apps/api && yarn 
```


#### How run project

```
yarn dev
```
